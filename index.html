How to Use This Template

1. Save the File

Copy this code and save it as index.html in the root folder of your GitHub repository.

2. Enable Pages

Go to your repo Settings > Pages, and select the 'main' branch and the '/ (root)' folder as the source.

3. Customize and Deploy

Edit the content, images, and colors right in this file! Tailwind makes styling fast and efficient.
